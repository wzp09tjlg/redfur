#### 三.思想
     1.一门技术出来，都是为了解决某个问题而存在的。所以在看技术的同时需要关注自己的问题点在什么地方。
     2.所有技术都存在有点和缺点。在日常开发过程中需要思考 为什么存在缺点，这些缺点是否能够有好的方案进行解决。
     3.技术是个过程，不断的总结和提炼才是正常之路。
